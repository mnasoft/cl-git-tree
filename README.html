1. Описание

cl-git-tree — инструмент для управления git‑локациями и синхронизации репозиториев, написанный на Common Lisp. Он позволяет регистрировать шаблоны локаций, строить URL для репозиториев, автоматизировать операции clone/unclone и управлять удалёнными репозиториями.

2. Возможности

  • Регистрация и хранение локаций в глобальной таблице locations.
  • Построение git‑URL на основе шаблонов.
  • Поддержка локальных путей, UNC и архивов (.tar, .xz).
  • Набор команд для работы с репозиториями:
    • clone / unclone
    • pull / push / commit / add
    • remote‑add / remote‑remove / remote‑readd
    • remake‑xz, info, aliases

3. Архитектура

  • cl-git-tree — основная система, объединяющая подсистемы и команды.
  • cl-git-tree/loc — подсистема для работы с локациями (<location>, locations, API).
  • cl-git-tree/git-utils — вспомогательные функции для работы с git.
  • src/commands/ — набор CLI‑команд.
  • src/fs/ — подсистема для работы с файловой системой.

4. Установка

(ql:quickload "cl-git-tree")

5. Использование

Пример регистрации локации и клонирования репозитория: 

(cl-git-tree/loc:add-location "gh"
  :name "GitHub"
  :url-git "https://github.com/")

(cl-git-tree/loc:repo-url "gh" "cl-git-tree")
;; => "https://github.com/cl-git-tree.git"
